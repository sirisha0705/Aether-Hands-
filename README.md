# AetherHands

A simple web camera effect that lets you control elements with your hands!



## Project Overview
This project showcases a responsive and user-friendly interface built using standard web technologies. 
It is intended as a learning-focused and portfolio-ready application, following basic best practices in project structure and version control.



## Technologies Used
- HTML5  
- CSS3  
- JavaScript  
- Git & GitHub

## How to Use
- Open index.html in your web browser
- Allow access to your webcam.
- Show your hands to the camera.

## Controls
Press these number keys on your keyboard to switch powers:

1: Fire & Ice (Left hand Fire, Right hand Ice)
Note: Painting with this power starts after a 10-second delay.

2: Lightning (Summons lightning bolts).

3: Red & Blue (Left hand Red, Right hand Blue)
Note: You cannot paint with this power.
